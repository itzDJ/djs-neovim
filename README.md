# DJ's custom nvim
## Plugin list
- packer
- copilot
- mason
- mason-lspconfig
- nvim-lspconfig
- vim-commentary
- vim-visual-multi
- markdown-preview
- lualine
- telescope
- github-nvim-theme
- nvim-treesitter
- harpoon
- alpha-nvim

## Install
`git clone https://github.com/itzDJ/djs-neovim`
`./install.sh`

## Uninstall
`rm -rf ~/.config/nvim`
`rm -rf ~/.local/share/nvim`
