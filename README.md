# DJ's custom nvim
## Plugin list (and dependencies)
- packer
- copilot (node.js)
- mason (git & curl)
- mason-lspconfig
- nvim-lspconfig
- vim-commentary
- vim-visual-multi
- markdown-preview
- lualine
- telescope (ripgrep?)
- github-nvim-theme
- nvim-treesitter
- harpoon
- alpha-nvim

## Install
```
git clone https://github.com/itzDJ/djs-neovim
./install.sh
```

## Uninstall
```
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
```
